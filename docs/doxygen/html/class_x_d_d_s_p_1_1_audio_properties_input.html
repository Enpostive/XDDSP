Detailed Description

template<int TimeMode, int ChannelCount = 1>
class XDDSP::AudioPropertiesInput< TimeMode, ChannelCount >

A coupler for producing signals based on a chosen DSP Parameter (such as sample rate or song tempo) This coupler is similar to ControlConstant, except that the control signal is taken from the Parameters Object and scaled by a multiplier.

Template Parameters
TimeModeSee AudioPropertiesInputMode for a list of available modes.
ChannelCountThe number of channels to be made available (all channels output the same signal).
