Detailed Description

Mixing Laws.

Each struct here contains a function called getWeights which returns a pair of mix weights suitable for use to mix or pan a pair of signals according to a parameter. Expected inputs for getWeights are between 0 and 1 Use these classes for Crossfader and Panner components
