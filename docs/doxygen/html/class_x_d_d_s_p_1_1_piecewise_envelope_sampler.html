Detailed Description

template<typename PositionIn>
class XDDSP::PiecewiseEnvelopeSampler< PositionIn >

Samples a piecewise envelope, using an input to control what part of the envelope is sampled.

The module connects to an XDDSP::PiecewiseEnvelopeData to get the information for the envelope.

Template Parameters
PositionInCouples to a position input.
