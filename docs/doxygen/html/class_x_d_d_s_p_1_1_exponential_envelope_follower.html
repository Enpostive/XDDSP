Detailed Description

template<typename SignalIn, typename RiseIn, typename FallIn, int StepSize = INT_MAX>
class XDDSP::ExponentialEnvelopeFollower< SignalIn, RiseIn, FallIn, StepSize >

Creates an exponential envelope signal from an input signal.

Attack and release times are expected in samples.

Template Parameters
SignalInCouples to the input signal.
RiseInCouples to the signal controlling the rise time.
FallInCouples to the signal controlling the fall time.
StepSizeControl how often the rise/fall times are updated. Leave at default for static signals.
