Detailed Description

template<int SignalCount = 1, int DefaultRamp = 0>
class XDDSP::RampTo< SignalCount, DefaultRamp >

A component for ramping from a current value to a new value.

This is designed so that it can be swapped in place of a ControlConstant in most cases (no modifier support)

Template Parameters
SignalCountThe number of channels to make available.
DefaultRampThe length of a ramp in samples if no ramp time is specified.
