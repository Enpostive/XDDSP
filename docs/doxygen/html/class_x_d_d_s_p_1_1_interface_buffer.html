Detailed Description

template<typename SignalIn>
class XDDSP::InterfaceBuffer< SignalIn >

A component which buffers samples from its input and makes them available in a thread safe manner.

Template Parameters
SignalInCouples to the signal to monitor. Can have as many channels as you like.
