Detailed Description

template<unsigned int MaxPoints = 10, int CurveResolution = 5>
class XDDSP::PiecewiseEnvelopeData< MaxPoints, CurveResolution >

A class containing a data structure to describe a piecewise envelope and code to synthesise the envelope.

Template Parameters
MaxPointsThe maximum number of points allowed.
CurveResolutionThe number of individual segments a curve is divided up into.

SampleType value,
SampleType curve )
inline

Change a point on the curve.

When a point change is requested, the change may or may not be constrained, depending on the setting given in PiecewiseEnvelopeData::setContrainEdits. When set to true, points cannot be moved to go before the point before it or after the point after it. When set to false, the points are reordered as necessary to keep the points in time order. The index of the changed point might change, so the caller is returned the index of the point after chages are made.

Parameters
indexThe index of the point to change.
timeThe new time of the point.
valueThe new value of the point.
curveThe new curve of the point.
Returns
int The new index of the point, or the old one if the index didn't change.
