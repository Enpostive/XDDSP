Detailed Description

template<typename SignalIn>
class XDDSP::SignalProbe< SignalIn >

A probe suitable for measureing minimum, maximum and instantaneous values in a signal and reading them from non-DSP code.

This component is thread safe.

Template Parameters
SignalInCouples to the signal to measure. May have as many channels as you like.
