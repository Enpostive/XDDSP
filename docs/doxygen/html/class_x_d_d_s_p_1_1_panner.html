Detailed Description

template<typename SignalIn, typename PanIn, typename MixLaw, int StepSize = 16>
class XDDSP::Panner< SignalIn, PanIn, MixLaw, StepSize >

A component for panning an input signal between two output signals.

Template Parameters
SignalInCouples to an input. May have as many channels as you like.
PanInCouples to the pan signal. Must have either one channel, or the same number of channels as the input.
MixLawA class from the MixingLaw namespace to configure the panner.
StepSizeSets how often the component updates the mix levels from MixLaw. By default this is once every 16 samples.
