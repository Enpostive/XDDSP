Detailed Description

template<int BufferCount>
class XDDSP::OutputBuffer< BufferCount >

An implementation of a buffer to be used to store output data from a DSP process. Instead of inheriting the coupler code above, a similar interface is presented which returns references to the samples requested, enabling them to be written by DSP code.

Template Parameters
BufferCountThe number of channels to support
