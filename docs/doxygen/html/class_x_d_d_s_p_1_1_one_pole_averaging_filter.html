Detailed Description

template<typename SignalIn>
class XDDSP::OnePoleAveragingFilter< SignalIn >

A component encapsulating a simple one-pole averaging filter, suitable for use as a simple lowpass filter, control smoother, RMS filter etc.

Template Parameters
SignalInCouples to the input signal. Can have as many channels as you like.
