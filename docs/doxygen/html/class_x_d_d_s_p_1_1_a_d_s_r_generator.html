Detailed Description

template<typename AttackTimeSamples, typename DecayTimeSamples, typename SustainLevel, typename ReleaseTimeSamples, int StepSize = IntegerMaximum>
class XDDSP::ADSRGenerator< AttackTimeSamples, DecayTimeSamples, SustainLevel, ReleaseTimeSamples, StepSize >

A component for generating ADSR envelopes.

Template Parameters
AttackTimeSamplesCouples to the signal specifying the attack time in samples. Only one channel is allowed.
DecayTimeSamplesCouples to the signal specifying the decay time in samples. Only one channel is allowed.
SustainLevelCouples to the signal to use for the sustain level. Only one channel is allowed.
ReleaseTimeSamplesCouples to the release time in samples. Only one channel is allowed.
StepSizeThe step size to use if the attack, delay or release samples are dynamic. Only one channel is allowed.
