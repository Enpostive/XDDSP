Detailed Description

template<typename Source, int ChannelCount = 1>
class XDDSP::SignalModifier< Source, ChannelCount >

A signal modifier coupler which applies a specified function to it's input. This coupler is meant to be a convenient way to apply a simple stateless function, such as a waveshaper, to a signal. The function may be called multiple times for any sample and may be called out of order.

Template Parameters
SourceThe class of the input coupler.
ChannelCountThe number of channels to make available.
