Detailed Description

template<typename SignalIn>
class XDDSP::DynamicsProcessingGainSignal< SignalIn >

Takes an envelope signal as input and produces a gain signal for dynamics control.

This component produces a gain signal in response to an envelope signal, according to the specified dynamics settings.

Template Parameters
SignalInCouples with the input signal.
