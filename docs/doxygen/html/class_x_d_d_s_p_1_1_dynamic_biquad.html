Detailed Description

template<typename SignalIn, typename FreqIn, typename QIn, typename GainIn, int StepSize = 16>
class XDDSP::DynamicBiquad< SignalIn, FreqIn, QIn, GainIn, StepSize >

A component containing a dynamic biquad filter.

This filter hides its configuration object, instead opting to configure the filter based on input signals provided, eg. signals for frequency and quality.

Template Parameters
SignalInCouples to the input signal. Can have as many channels as you like.
FreqInCouples to the signal providing the input frequency in Hz. Must have only one channel.
QInCouples to the signal providing the quality factor. Must have only one channel.
GainInCouples to the signal specifying the gain in dB. Must have only one channel.
StepSizeControls how often the component reconfigures the filter from the inputs. Default is every 16 samples.
