Detailed Description

template<int ConstantCount = 1>
class XDDSP::ControlConstant< ConstantCount >

A coupler which outputs a constant signal on each channel. A modifier function may be given to modify given control values before they enter the DSP network.

Template Parameters
ConstantCountThe number of channels to make available.
