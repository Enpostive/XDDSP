Detailed Description

template<typename SignalIn, unsigned long ReserveSize = 32768>
class XDDSP::LUFSBlockCollector< SignalIn, ReserveSize >

Measures RMS levels and reports the overall loudness as described in the LUFS standard. This component only does the block splitting and RMS measurement parts of the standard. A k-Weighted filter is required to be placed before this component in the signal chain to obtain an actual dBLUFS measurement.

Template Parameters
SignalInCouples to the input signal.
ReserveSizeThe size to reserve for the first memory allocation.
