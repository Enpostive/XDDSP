Detailed Description

template<int ChannelCount = 2>
class XDDSP::PluginInput< ChannelCount >

A coupler which is convenient for connecting as the input for the entire network. Simple bounds checking is provided in debug mode to alert the developer if samples outside of the buffer are being requested.

Template Parameters
ChannelCountThe number of channels to make available.
